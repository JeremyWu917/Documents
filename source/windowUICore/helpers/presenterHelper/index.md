# SetOverlappedPresenterState
```cs
PresenterHelper.SetOverlappedPresenterState(window, OverlappedPresenterState.Maximized);
```

# SetOverlappedPresenter
```cs
PresenterHelper.SetOverlappedPresenter(window, overlappedPresenter);
```
